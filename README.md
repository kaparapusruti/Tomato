# Food Delivery Web App

## Project Description
This is a full-stack food delivery web application that allows users to browse food items, add them to cart, and place orders securely. The application includes authentication and a responsive user interface.

## Features
- User authentication (Login / Signup)
- Browse food menu
- Add to cart functionality
- Order management
- Responsive UI

## Tech Stack
Frontend: React  
Backend: Node.js, Express  
Database: MongoDB  
Authentication: JWT  

## How to Run Locally
1. Clone the repository
2. Install dependencies
3. Start backend and frontend servers
